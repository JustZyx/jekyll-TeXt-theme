# justzyx.github.io
> The blog for zyx, also a field for technical experiment. | http://blog.justzyx.cn

## up and running
```
git clone https://github.com/JustZyx/JustZyx.github.io.git
bundle install --path vendor/bundle
bundle exec jekyll serve
```
